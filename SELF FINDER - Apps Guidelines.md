# SELF FINDER Brand Guidelines

## Brand Name
**SELF FINDER**


### Acceptable Usage
- SELF FINDER (all caps)

### Unacceptable Usage
- Self Finder (Title Case)
- self finder (all lowercase)
- SELFFINDER (no space)

---

## Logo Usage

### Clear Space
Maintain clear space around the logo equal to the height of the Star icon in the logo on all sides.

### Minimum Size
- Digital: 120px width minimum
- Print: 1 inch / 25mm width minimum

### Logo Don'ts
- Do not stretch or distort the logo
- Do not rotate the logo
- Do not change the logo colors outside of approved variations
- Do not add effects (shadows, gradients, outlines)
- Do not place the logo on busy backgrounds

---

## Color Palette

### Primary Brand Colors

**Deep Navy**
- HEX: #211D52
- RGB: 33, 29, 82
- CMYK: 95, 93, 31, 26
- Use as: Primary brand color, main headers, primary buttons, logo ring

**Ocean Teal**
- HEX: #12796A
- RGB: 18, 121, 106
- CMYK: 85, 29, 60, 13
- Use as: Key visual accents, secondary buttons, success states

### Secondary Accent Colors

**Golden Star**
- HEX: #F6A02D
- RGB: 246, 160, 45
- CMYK: 2, 41, 93, 0
- Use as: Highlights, notification dots, star icon accents

**Wave Blue Medium**
- HEX: #5B79C7
- RGB: 91, 121, 199
- CMYK: 65, 45, 0, 0
- Use as: Mid-level background illustrations, hover states

**Sky Blue Light**
- HEX: #CBE6F7
- RGB: 203, 230, 247
- CMYK: 18, 4, 0, 0
- Use as: Soft background sections, informational callout boxes

### Neutral Colors

**White**
- HEX: #FFFFFF
- Use as: Backgrounds, text on dark backgrounds, cards

**Light Gray**
- HEX: #F5F7FA
- Use as: Page backgrounds, subtle dividers

**Dark Gray**
- HEX: #4A4A4A
- Use as: Body text, secondary text

**Black**
- HEX: #1A1A1A
- Use as: Headings, primary text (use sparingly in favor of Deep Navy)

## Typography

### Primary Font
**Montserrat** (or Helvetica Neue, Arial as fallbacks)

### Font Weights
- **ExtraBold (800) / Bold (700)**: Main Headers, primary emphasis
- **Semi-Bold (600)**: Subheadings, UI labels, button text
- **Medium (500) / Regular (400)**: Body text, paragraphs, standard descriptions
- **Light (300)**: Captions, secondary text (use sparingly)

### Font Sizes
- H1: 32px / 2rem
- H2: 24px / 1.5rem
- H3: 20px / 1.25rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Caption: 12px / 0.75rem

### Line Height
- Headings: 1.2
- Body: 1.5

---

## UI Elements

### Buttons

**Primary Button**
- Background: Deep Navy (#211D52)
- Text: White (#FFFFFF)
- Border Radius: 8px
- Hover: Ocean Teal (#12796A)

**Secondary Button**
- Background: Transparent
- Border: 2px solid Deep Navy (#211D52)
- Text: Deep Navy (#211D52)
- Hover: Sky Blue Light (#CBE6F7) background tint

### Cards
- Background: White (#FFFFFF)
- Border Radius: 12px
- Shadow: 0 2px 8px rgba(0, 0, 0, 0.08)

### Input Fields
- Border: 1px solid #E0E0E0
- Border Radius: 8px
- Focus Border: Deep Navy (#211D52)

---

## Iconography

- Style: Outlined or filled, consistent stroke width
- Size: 24px standard, 20px small, 32px large
- Color: Match text color or use Deep Navy (#211D52) / Ocean Teal (#12796A) for active/emphasis states

---

## Tone of Voice

- **Exploratory & Guiding**: Encouraging discovery and personal growth
- **Professional**: Clear, trustworthy, knowledgeable
- **Approachable**: Friendly, helpful, not overly formal
- **Concise**: Direct, no jargon, easy to understand
